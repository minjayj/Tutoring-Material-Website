Tutor's Study Guide: Unit 1 Using Objects & Methods

Primitive Types vs. Reference Types:

  • Primitives (int, double, boolean) store the actual numerical or boolean value directly in memory.
  • Reference types (Objects like String) store the memory address (reference) where the object data is located.

Essential String Methods (Strings are Immutable!):

  • length(): Returns the total number of characters.
  • substring(int from, int to): Extracts characters starting at index from up to (but not including) index to.
  • indexOf(String str): Returns the index of the first occurrence of str. Returns -1 if not found.

The Math Class (Static Methods): Because these are static, you call them directly on the class name without creating a Math object. Know how to use Math.abs(), Math.pow(base, exponent), Math.sqrt(), and Math.random() (which generates a double from 0.0 inclusive up to 1.0 exclusive).
